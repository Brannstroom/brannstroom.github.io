Hva er semantiske tagger og hvorfor er de viktig?

13/09-2020

Semantiske tagger er nytt i HTML5 og er tagger med betydning. Semantikk handler veldig enkelt om betydningen av ordet og i HTML brukes for å beskrive hva innholdet vil være til utvikleren og nettleseren. Før HTML5 ble det brukt det man nå kaller for ikke-semantiske tagger som f.eks. div og span for så å kategorisere de ved å bruke id og class i selve taggene.

Noen eksempler på de nye semantiske taggene i HTML5 er header, footer og form. I motsetning til de gamle div og span taggene som man tidligere bruke, har disse semantiske taggene veldig spesifike bruksområder som man aldri bør bryte. Som en mennesklig besøker av nettsiden vil ikke samantiske tagger endre min opplevelse, men ettersom datamaskiner er dumme og kun klarer å gjøre det vi ber de, så er det viktig å bruke taggene, slik at crawlers fra søkemotorer enklere og kjappere kan lese sidens innhold. Dette hjelper veldig med SEO (Search Engine Optimiztion), noe som vil gjøre at din side kommer høyere opp på søkemoterer som google.
